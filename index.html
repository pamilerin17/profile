About Me

My Journey

I'm a passionate frontend developer who started my coding journey during college. What began as a curiosity about how websites work has evolved into a deep love for creating intuitive, accessible user experiences. Over the past five years, I've worked with various technologies and frameworks, but my core philosophy remains the same: build things that work well for everyone.

Goals in This Program

Technical Mastery

I aim to deepen my understanding of modern JavaScript patterns and improve my skills in building scalable, maintainable applications.

Accessibility Advocacy

I want to become a stronger advocate for web accessibility, ensuring the digital products I build are usable by everyone regardless of ability.

Collaboration Skills

I hope to enhance my ability to work effectively in team environments and contribute to open source projects.

Areas of Low Confidence

Advanced State Management

While comfortable with basic state management, I sometimes struggle with complex state patterns in large applications.

Performance Optimization

Identifying and fixing performance bottlenecks, especially in React applications, is an area I'm actively working to improve.

Backend Integration

While I understand REST APIs well, I want to gain more confidence working with GraphQL and real-time data streams.

Note to My Future Self

Remember why you started this journey - because you love creating things that help people. Don't get so caught up in the latest frameworks that you forget the fundamentals. Keep learning, stay curious, and always prioritize the human using your software over the technology you're using to build it.

Additional Reflections

The web development landscape changes rapidly, but the core principles of good design and user-centered development remain constant. I'm excited to continue growing as a developer while maintaining focus on building inclusive, performant web experiences.

I believe that great software is built through collaboration, continuous learning, and a willingness to adapt. Every project is an opportunity to improve and every challenge is a chance to learn something new.
